#  GameHub

GameHub is a Java desktop application that brings multiple mini games together in one place. Instead of opening different programs for different games, users can simply log in and choose any game from a single platform.

The main goal of this project is to create a fun and interactive gaming environment while also demonstrating concepts like Java GUI development, game logic, and database connectivity.

This project was developed as part of our academic work to explore how multiple technologies can be combined to build a complete application.

---

##  Features

* User login system
* Multiple games in one application
* Simple and interactive interface
* Central game launcher menu
* Database support for storing user information and scores

---

##  Games Available

GameHub currently includes the following mini games:

**Maze Escape**
Players navigate through a maze and try to reach the exit without getting stuck.

**Whack-a-Mole**
A reflex game where players quickly hit the moles appearing in a 3×3 grid using number keys.

**Reaction Time Tester**
Tests how fast a player reacts when a signal appears on the screen.

**Flappy Pig**
Guide the pig through obstacles while trying to survive as long as possible.

**Syntax Snake**
A variation of the classic snake game with a coding-themed twist.

**Cake Maker**
A fun game where players create cakes by choosing the correct ingredients.

---

##  Technologies Used

* **Java** – Main programming language
* **Java Swing / AWT** – Used for creating the graphical user interface
* **SQL Database** – Stores user details and scores
* **JDBC** – Connects the application with the database

---

##  Project Structure

The GameHub application consists of several main components:

**Login System**
Handles user authentication and allows players to access the platform.

**Game Launcher**
The main menu where users can select which game they want to play.

**Game Modules**
Each game is implemented as a separate Java class within the project.

**Database**
Used to store user accounts and keep track of player scores.
